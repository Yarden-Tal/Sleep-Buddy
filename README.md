# Sleep Buddy

**A minimalistic white noise Flutter app** by [Yarden Tal](https://github.com/Yarden-Tal).
This app features: 
- Several different white noise sounds for sleep or focus
- A timer for stopping the selected sound
- Dark & Light mode
